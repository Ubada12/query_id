Welcome to the Queries API!

Use the following endpoints to fetch or refresh the queries:

Fetch Endpoints:

  1. Get all queries from the database
    GET /api/getAll/query
  2. Get all queries with the specified user ID
    GET /api/getAll/query?userid=[user_id]
  3. Get all queries with the specified bot name
    GET /api/getAll/query?bot=[bot_name]
  4. Get all queries with the specified user ID and bot name
    GET /api/getAll/query?userid=[user_id]&bot=[bot_name]

Refresh Endpoints:

  1. Refresh and update all queries
    POST /api/refreshAll/query
    Body: {}
  2. Refresh and update all queries with the specified user ID
    POST /api/refreshAll/query
    Body: { "userid": "user_id" }
  3. Refresh and update all queries with the specified bot name
    POST /api/refreshAll/query
    Body: { "bot": "bot_name_value" }
  4. Refresh and update all queries with the specified user ID and bot name
    POST /api/refreshAll/query
    Body: { "userid": "user_id", "bot": "bot_name_value" }
